- React
- Axios

# Requisitos

- npm install
- npm install axios

# Para rodar o projeto

- Clone esse repositório
- Faça o CD para a pasta clonada
- Abra o terminal
- npm install
- npm install axios
- npm start

O servidor deve rodar no http://localhost:3000/
